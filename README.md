Netflix User Analytics

Overview

This project analyzes Netflix user data and applies Machine Learning techniques to understand user behavior and predict subscription renewal.

Dataset Features

- UserID
- Age
- Gender
- SubscriptionType
- WatchHoursPerWeek
- DevicesUsed
- FavoriteGenre
- AdClicks
- MonthlySpend
- SubscriptionRenewed

Tasks Performed

Dataset Understanding

- Loaded and explored the dataset
- Identified numerical and categorical features
- Checked for missing values

Exploratory Data Analysis

- Calculated average age, watch hours, and monthly spending
- Analyzed subscription categories
- Determined subscription renewal percentage

Machine Learning Models

- Decision Tree Classification
- K-Nearest Neighbors (KNN)
- Linear Regression

Objectives

- Predict whether a user will renew their subscription
- Compare classification algorithms
- Predict monthly spending of users
- Gain business insights from user behavior data

Tools and Libraries

- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab
